Description

ObjectStore - this defines the object-store (Cloud Files) service.

Usage:

     $conn = new OpenStack('{URL}', '{SECRET}');
     $ostore = new OpenCloud\ObjectStore(
         $conn,
         'service name',
         'service region',
         'URL type'
     );

Default values for service name, service region, and urltype can be provided via the global constants RAXSDK_OBJSTORE_NAME, RAXSDK_OBJSTORE_REGION, and RAXSDK_OBJSTORE_URLTYPE.
